Click the button to open Settings with Notifications tab

Manual Steps:

  1. Login to the application at http://localhost:3838
  2. Navigate to Settings (or use the button above)
  3. Click on the "Notifications" tab
  4. Look for "Smart Notification Features" section
  5. Check if the checkboxes have the card-based design with orange gradients

Expected Result:

You should see 3 checkboxes with:

  • Card-based design with padding and borders
  • Orange gradient when checked
  • Title and description for each option
  • Hover effects
